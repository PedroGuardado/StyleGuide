Fonts

For H1 Heading, please use Roboto Mono with a font weight of Bold and size of 65px.

For H2 Subheding, please use regular Roboto Mono with a font style of Italic and size of 35px

For P Paragraph, please use regular Sarabun with no text styling, font size of 20px

For Footer, please use regular Old Standard TT with font size of 15px
